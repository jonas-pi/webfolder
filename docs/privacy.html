隐私政策

应用名称：网址收藏夹（WebBookmarks）

本政策说明本应用在您设备上如何处理信息。使用本应用即表示您理解并同意本政策内容。

1. 我们收集的信息

本应用不以盈利为目的收集您的个人信息，也不内置广告跟踪。与您使用相关的主要数据包括：

  • 书签与收藏数据：您输入或导入的标题、网址、文件夹结构等，仅保存在本应用在您设备上的私有目录中。
  • 应用设置：如默认搜索引擎、搜索框位置、界面相关偏好等，以本地文件形式存储在设备上。
  • 隐私同意状态：您是否已同意本政策，以设备本地首选项（Preferences）形式记录，用于首次启动及「我的」中的隐私确认流程；该记录不会上传至开发者服务器。
  • 可选的壁纸与图标缓存：若您选择背景图片或展示网站图标，相关文件或缓存可能保存在设备本地；网站图标可能通过公开网络请求获取对应站点资源。

2. 首次使用同意、设置内查阅与撤回

  • 首次打开：您需通过应用内弹窗阅读摘要，并可点击《隐私政策》在系统浏览器中打开本页面查看全文。只有选择「同意」后，方可使用本应用主要功能；选择「不同意」或等效取消操作时，应用将退出，且不会在本地写入「已同意」标记。
  • 「我的」—「隐私政策」：您可随时再次打开同一形式的确认对话框，阅读摘要、在浏览器中查看本页全文，并再次选择「同意」或「不同意」。若选择「不同意」，应用将清除设备上已保存的同意记录，使您在下一次启动时再次经历首次同意流程，且本应用会立即退出。若您仅通过系统返回键等方式关闭该对话框而未点选「不同意」，则不视为撤回同意，已保存的同意状态保持不变。
  • 自愿继续使用：在设置内选择「同意」表示您确认继续接受本政策，通常无需重复写入数据（您此前已同意时状态保持不变）。

3. 权限说明

  • 网络访问（INTERNET）：用于在浏览器中打开您选中的书签链接、按需获取网站图标、在确认流程中打开本隐私政策网页，以及在您主动配置并操作时与 GitHub 等第三方服务通信（例如可选的 Gist 同步）；不会在您未操作相关功能时由本应用向开发者服务器上传您的书签内容。
  • 振动（VIBRATE）：用于在点击、切换等操作时提供轻微触感反馈，不涉及个人数据。

4. 数据存储与安全

书签与设置默认保存在本机应用沙箱内，由系统隔离保护。除您主动使用「导入」等由您选择的文件操作外，应用不会将您的书签列表自动上传到由开发者运营的服务器（本应用未要求您向开发者提供此类账号）。

若您使用云端同步（GitHub Gist）功能，由您在应用内自行填写 Token、Gist 等信息，书签数据由您的设备与 GitHub 之间直接传输，适用 GitHub 的条款与隐私政策；请您妥善保管访问令牌，勿与他人分享。

5. 第三方服务与链接

本应用会按您的操作打开您选择的第三方网站，或在您确认后打开本政策托管页面（例如 GitHub Pages）。这些网站有其各自的隐私政策，本应用无法控制，请您在访问时留意。

6. 未成年人

若您为未成年人，请在监护人指导下使用本应用。

7. 政策更新

我们可能随应用版本更新修订本政策。更新后的版本将在本页面或通过应用内适当方式提示，请定期查阅。若修订涉及处理方式的重大变化，我们将通过合理方式提醒您，必要时可能要求您重新确认同意。
